Exercise 4.

Para cada una de las siguientes funciones racionales, identifica la ubicación de cualquier posible hueco en el gráfico. Luego, crea una tabla de valores de la función para valores de entrada cercanos a donde debería estar el hueco. Usa tu trabajo para decidir si el gráfico realmente tiene un hueco, con justificación escrita.
  1. \(\displaystyle \displaystyle r(x) = \frac{x^2-16}{x+4}\)
  2. \(\displaystyle \displaystyle s(x) = \frac{(x-2)^2(x+3)}{x^2 - 5x - 6}\)
  3. \(\displaystyle \displaystyle u(x) = \frac{(x-2)^3(x+3)}{(x^2 - 5x - 6)(x-7)}\)
  4. \(\displaystyle \displaystyle w(x) = \frac{x^2 + x - 6}{(x^2 + 5x + 6)(x+3)}\)
  5. Verdadero o falso: dado \(r(x) = \frac{p(x)}{q(x)}\text{,}\) si \(p(a) = 0\) y \(q(a) = 0\text{,}\) entonces \(r\) tiene un hueco en \(x = a\text{.}\)
