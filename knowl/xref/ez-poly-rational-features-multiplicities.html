Exercise 9.

For each of the following rational functions, identify the location of any potential hole in the graph. Then, create a table of function values for input values near where the hole should be located. Use your work to decide whether or not the graph indeed has a hole, with written justification.
  1. \(\displaystyle \displaystyle r(x) = \frac{x^2-16}{x+4}\)
  2. \(\displaystyle \displaystyle s(x) = \frac{(x-2)^2(x+3)}{x^2 - 5x - 6}\)
  3. \(\displaystyle \displaystyle u(x) = \frac{(x-2)^3(x+3)}{(x^2 - 5x - 6)(x-7)}\)
  4. \(\displaystyle \displaystyle w(x) = \frac{x^2 + x - 6}{(x^2 + 5x + 6)(x+3)}\)
  5. True or false: given \(r(x) = \frac{p(x)}{q(x)}\text{,}\) if \(p(a) = 0\) and \(q(a) = 0\text{,}\) then \(r\) has a hole at \(x = a\text{.}\)
