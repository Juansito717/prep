Exercise 3.

Hemos visto que cualquier función exponencial \(f(t) = b^t\) (\(b \gt 0\text{,}\) \(b \ne 1\)) se puede escribir en la forma \(f(t) = e^{kt}\) para algún número real \(k\text{,}\) y esto es porque \(f(t) = b^t\) es una escala horizontal de la función \(E(t) = e^{t}\text{.}\) En este ejercicio, exploramos cómo el logaritmo natural se puede escalar para lograr un logaritmo de cualquier base.
Sea \(b \gt 1\text{.}\) Debido a que la función \(y = f(t) = b^t\) tiene una función inversa, tiene sentido definir su inversa como lo hicimos cuando \(b = 10\) o \(b = e\text{.}\) El logaritmo en base \(b\), denotado \(\log_b(y)\) se define como la potencia a la que elevamos \(b\) para obtener \(y\text{.}\) Así, escribir \(y = f(t) = b^t\) es lo mismo que escribir \(t = f^{-1}(y) = \log_b(y)\text{.}\)
En Desmos, la función del logaritmo natural se da por ln(t), mientras que el logaritmo en base \(10\) por log(t). Para obtener un logaritmo de una base diferente, como un logaritmo en base \(2\text{,}\) escribe log_2(t) (el guion bajo generará un subíndice; luego usa la flecha derecha para salir del modo subíndice).
En una nueva hoja de trabajo de Desmos, ingresa V(t) = k * ln(t) y acepta el deslizador para \(k\text{.}\) Establece los límites inferior y superior para el deslizador en \(0.01\) y \(15\text{,}\) respectivamente.
  1. Define \(f(t) = \log_2(t)\) en Desmos. ¿Puedes encontrar un valor de \(k\) para el cual \(\log_2(t) = k\ln(t)\text{?}\) Si es así, ¿cuál es el valor? Si no, ¿por qué no?
  2. Repite (a) para las funciones \(g(t) = \log_3(t)\text{,}\) \(h(t) = \log_5(t)\text{,}\) y \(p(t) = \log_{1.25}(t)\text{.}\) ¿Qué patrón(es) observas?
  3. Verdadero o falso: para cualquier valor de \(b \gt 1\text{,}\) la función \(\log_b(t)\) se puede ver como una escala vertical de \(\ln(t)\text{.}\)
  4. Calcula los siguientes valores: \(\frac{1}{\ln(2)}\text{,}\) \(\frac{1}{\ln(3)}\text{,}\) \(\frac{1}{\ln(5)}\text{,}\) y \(\frac{1}{\ln(1.25)}\text{.}\) ¿Qué notas sobre estos valores en comparación con los de \(k\) que encontraste en (a) y (b)?
